<h2> A single line comment in C++ language source code can begin with _____ </h2>
</div>
<pre><code class="language-cpp">

</code></pre>

ОТВЕТЫ:
1) ;
2) :
3) /*
4) //

<br>
<br>
ПРАВИЛЬНЫЙ ОТВЕТ:
4) //


<br> <br>
ОТВЕТ РАЗВЕРНУТЫЙ:
Two immediate forward slashes are used to comment a single line. A single can
be commented by beginning with /* and should be terminated with */ , in
general used for multi-line comments.
